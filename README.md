# Lab Fabric Intro

This repository contains CSV-ready source files for an introductory Microsoft Fabric / Power BI lab.

The files come from the Microsoft Power BI workshop source data:

https://github.com/microsoft/pbiworkshops/tree/main/Day%20After%20Dashboard%20in%20a%20Day/Source_Files

## Files

The `source-files/` folder contains:

- Dimension files converted from the original pipe-delimited CSV files to comma-delimited UTF-8 CSV.
- `FactOnlineSales_1.csv` through `FactOnlineSales_9.csv`, converted from the original Parquet files to comma-delimited UTF-8 CSV.
- A `source-files/README.md` manifest with row counts, column counts, and original source URLs.

The original single-file export is also kept at the repository root:

- `FactOnlineSales_9.csv`
